# Free Course Conversion Design

> Strategy for the 5 free gateway courses — one per age group.
> Goal: maximise registered users and paid course conversions.

---

## Principles

1. **Value first** — free courses must be genuinely useful, not crippled demos.
2. **Natural cliff** — lock content after a meaningful milestone, not arbitrarily.
3. **Show the path** — locked lessons display title + description so learners know what they're missing.
4. **One clear CTA** — never two competing calls to action on the same screen.
5. **Personalised recommendations** — next course suggestion matches the learner's age group and completed skills.

## Course 1 — Wonder Lab: Science for Tiny Explorers (Ages 3–5)

### Teaser Strategy
- Modules 1–2 (6 lessons) fully unlocked for non-registered visitors.
- Module 3 (3 lessons: "Light & Shadow," "Mixing Colours," "Our Final Experiment") visible with lock icon.
- Each locked lesson shows title, duration, and a 1-sentence description.

### Registration Gate
- Triggered before Module 3 Lesson 1.
- Copy: *"You're doing great! Create a free account to unlock the rest of Wonder Lab — and save your progress."*
- Gate type: soft (registration, no payment required).

### Upgrade CTA
- Placement: end of the last free lesson + course completion screen.
- Copy: *"Your little scientist is ready for more! Continue the adventure with Little Coders Unplugged."*
- Button: **Explore Next Course** → `/courses/little-coders-unplugged`

### Next Paid Course Recommendation
- **Primary:** Little Coders Unplugged (Ages 3–5, Coding)
- **Secondary:** Tiny Engineers (Ages 3–5, Engineering)

### Bundle Recommendation
- *Ages 3–5 Explorer Bundle* — Wonder Lab + Little Coders + Tiny Engineers + Space & Sky + Creative Robot Stories
- Bundle copy: *"Complete the full preschool STEM journey. Save 30% with the Explorer Bundle."*

### Locked Content Handling
- Locked lessons: blurred thumbnail + lock icon + title + duration visible.
- No preview video for locked lessons (keep mystery, drive registration).

## Global Conversion Notes

| Signal | Action |
|---|---|
| User completes free course | Trigger upgrade CTA email within 1 hour |
| User views 2+ locked lessons | Show "Unlock Bundle" modal on next page load |
| User returns 3+ days after registration without enrolling | Re-engagement email with personalised course recommendation |
| User's child completes a free course (parent account) | Notify parent with next course recommendation |
| User watches >50% of a locked lesson preview | Add to high-intent segment; show discount offer next session |
